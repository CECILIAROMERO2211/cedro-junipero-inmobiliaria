# Cedro & Junípero - Inmobiliaria de Lujo

Plataforma inmobiliaria de alto nivel diseñada para la curaduría de propiedades exclusivas, centrada en la arquitectura, la privacidad y el estilo de vida.

## 🚀 Acerca del Proyecto
Este proyecto es una aplicación web SPA (Single Page Application) desarrollada como una solución inmobiliaria moderna y elegante. El objetivo fue crear una experiencia de usuario que refleje el "Lujo Silencioso", integrando navegación fluida, gestión dinámica de propiedades y un sistema de contacto funcional.

## 🛠 Tecnologías Utilizadas
- **Frontend**: React.js
- **Framework**: Vite
- **Estilos**: CSS puro (diseño a medida con enfoque en tipografía y espacios)
- **Navegación**: React Router DOM
- **Formularios**: EmailJS (para la gestión de citas y contactos en tiempo real)
- **Deployment**: Netlify

## ✨ Características Principales
*   **Diseño Responsivo**: Experiencia optimizada para escritorio y dispositivos móviles.
*   **Catálogo Dinámico**: Gestión de propiedades mediante Context API para mantener el estado global.
*   **Formulario Inteligente**: Envío de correos automatizado para captación de leads.
*   **SEO Optimizado**: Implementación de `react-helmet-async` para mejorar la visibilidad en buscadores y redes sociales.
*   **Curaduría Visual**: Uso de fuentes de alta gama (Cinzel, Playfair Display) y paleta de colores sofisticada.
